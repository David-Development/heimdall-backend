About this Prototype

This Prototype was implemented as Proof of Concept for the Masterthesis titled "Neuronale Netze und andere Verfahren zur Gesichtserkennung in der Heimautomatisierung" by Constantin Kirsch. It is based on the findings of a comparison of different Face Recognition Methods and can be used in a SmartHome environment. This means the server needs only local network access, only a few images for each person are needed and training takes place in a few minutes. For more information, please take a look at the sourcecode repository.
